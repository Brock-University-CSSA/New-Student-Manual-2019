    关于我们

    布鲁克大学中国学生学者联谊会

    Brock University Chinese Students and Scholars Association

    简称：BUCSSA

    2019届主席团成员：

          主席：许海智 （Jack）

          副主席：高璐，张佳怡（Jacqueline）

          秘书长：范美欣（Tanny）

          顾问：容艺盈，胡楚凡（Fannie）

    传媒部 ( Publicity Department )

            一个基础而又核心的部门，我们的职责是将各种信息进行分类，筛选，宣传后让学生能更快地掌握最新的资讯。作为社团对外发布消息的窗口，我们与各个部门紧密联系，努力提高着我们中国学联会的社会影响力。加入我们，你将得到个人文笔，文采上的提升，对于信息采集，归纳的能力加强，以及学习到各大社交平台后台管理 的流程步骤。笔头并不能拯救世界，但却可以记录历史

          传媒部长：黎彦均 (Emma) 熊一衡 (Florence)

    人事部 ( Human Resource Department )

            一个默默无闻，却又承担起构建整个学生会骨架的部门，我们的职责是将有限的社团人力资源，通过规 划分配，无限发挥每一个人的闪光点。加入我们，你将学习到如何更好地统筹有限的人力资源发挥1+1=3的效果，如何编排更有效率的活动人员分配 机制等等。

          人事部长：尹凯欣(Kate)

    设计部 ( Design Department )

            一个技术型人才汇聚的部门，从网站设计，编程，平面设计，3D建模，到实际物件设计，制作等，我们是站在社团设计方面的第一线人员。加入我们，你将会作为第一线设计人员参与社团所有的平面设计任务，以及成为每个活动中的摄影师和视频剪辑人员。我们将会培养你对于平面设计类软件的应用及操作，提升你将想法具现化的动手能力；也会培养你作为网站后台运营团队的职业素质。我们是一群热爱技术以及设计的部门，用实打实的技术征服在场的所有人

          设计部长： 周天羽 (Krunk) 赵思远 (Barry)

    财务部 ( Finance Department )

            一个跟社团资金打交道的部门，加入我们，你能够学习到如何整理各项活动预算总表，如何判断最优化采购方案以及如何为社团活动招募资金。财务部的存在以及其正常的运作是BUCSSA活动运作的基本保障

          财务部长：王欣悦 (Vicky)

    外联部 ( Public Relation Department )

            一个代表学生会整体形象的部门，我们以树立学校及学生会形象为主要任务，增强社会各界对布鲁克大学及中国学联会的了解和认可。加入我们，你将学习到如何挖掘潜力商家/赞助商，与商家建立良好的沟通基础以及合作关系，为社团活动的顺利举办提供充足的资源。在每一次与商家打交道的时候，你的个人语言组织能力会得到巨大的提升，使你更好的应对不同情景下的随机应变能力

          外联部长： 邢悦 (Echo)

    活动部 ( Event Department )

            一个全部门中头脑风暴能力最厉害的部门，在这里有天马行空的想象力，有说做就做的行动力，还有细致入微的观察力，以及坚不可摧的团队凝聚力。加入我们，你将学习到能将你的（脑洞）梦想一步步实现的必备技能，从想法的建立，到策划案的详细细节编写，再到紧张的活动布置现场，最后完美谢幕。你的活动组织能力能得到足够多的锻炼，并能收获一份珍贵的团队友谊

          活动部长：江文瑄 (Selina) 赵亦瑜 (Neil)
